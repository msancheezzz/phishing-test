¿Qué es el Phishing?

El phishing es un tipo de ciberataque que busca engañar a las víctimas para que revelen información sensible como contraseñas, datos bancarios o información personal.

Técnicas comunes

  • Correos electrónicos falsos que parecen legítimos
  • Sitios web que imitan a empresas conocidas
  • Mensajes urgentes que piden acción inmediata
  • Ofertas demasiado buenas para ser verdad

Señales de alerta

  • Errores gramaticales o ortográficos
  • Direcciones URL sospechosas
  • Solicitud de información personal
  • Presión para actuar rápidamente

¿Cómo protegerte?

Mejores prácticas de seguridad

  • Verifica siempre la URL antes de introducir credenciales
  • No hagas clic en enlaces de correos electrónicos sospechosos
  • Habilita la autenticación de dos factores (2FA)
  • Mantén tu software y navegadores actualizados
  • Usa contraseñas únicas y fuertes para cada cuenta

Herramientas útiles

  • Gestores de contraseñas (LastPass, Bitwarden, 1Password)
  • Autenticación de dos factores (Google Authenticator, Authy)
  • Navegadores con protección contra phishing
  • Software antivirus actualizado

¿Qué hacer si has sido víctima?

Si crees que has caído en un ataque real de phishing, sigue estos pasos:

  1. Cambia inmediatamente todas tus contraseñas
  2. Contacta con tu banco si has compartido información financiera
  3. Reporta el incidente a las autoridades correspondientes
  4. Informa a tus contactos si el ataque involucra tu correo o redes sociales
  5. Considera congelar tu crédito si se ha comprometido información sensible
